Go File & System Manager

Этот проект представляет собой веб-сервер на языке Go, предназначенный для управления файлами и выполнения системных команд через REST API.

Основные возможности:

Управление файлами: Загрузка, список и удаление файлов в указанной директории.

Системные операции: Выполнение команд обновления, резервного копирования и восстановления приложения/базы данных.

Конфигурация через окружение: Гибкая настройка портов, путей и команд через переменные среды.

Статический контент: Поддержка обслуживания статических файлов и главной страницы index.html.

Переменные окружения
Вы можете настроить работу сервера, используя следующие переменные окружения:

Переменная	            Описание	                            Значение по умолчанию

UPT_PORT	            Порт, на котором запустится сервер	    :8080
UPT_URL_API_PREFIX	                                            ""
UPT_PATH_PREFIX         Директория для загрузки файлов          ./uploads
UPT_LIMIT_DOWNLOAD_MB	Максимальный размер загрузки (МБ)	    500
UPT_SC_UPDATE	        Команда для обновления системы	        lscpu
UPT_SC_BACKUP_APP	    Команда бэкапа приложения	            who
UPT_SC_RESTORE_APP	    Команда восстановления приложения	    vmstat
UPT_SC_BACKUP_BD	    Команда бэкапа базы данных	            lsblk


API Эндпоинты:

Файловые операции
POST /api/upload — Загрузка одного или нескольких файлов.

GET /api/list — Получение списка файлов в директории загрузок (с размером и датой).

POST /api/delete — Удаление файла (требуется параметр filename).

Системные команды (GET запросы)
/api/update — Запуск команды обновления.

/api/backupAPP — Резервное копирование приложения.

/api/restoreAPP — Восстановление приложения.

/api/backupBD — Резервное копирование БД.
